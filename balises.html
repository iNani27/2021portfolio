Commentaires

<! Commentaire HTML -->

La balise de commentaire est utilisée pour insérer des commentaires dans le code source. Les commentaires ne sont pas affichés dans les navigateurs. Vous pouvez utiliser des commentaires pour expliquer votre code, ce qui peut vous aider lorsque vous modifiez le code source à une date ultérieure. Ceci est particulièrement utile si vous avez beaucoup de code.
